# Modal Popup - Vanilla JavaScript

A simple and accessible modal (popup) component built using "HTML, CSS, Js".
This project demonstrates common UI behaviour used in real-world web applications.

---------------------------------------

## Features
- Open modal using a button.
- Close modal using:
        1. Close (X) button.
        2. Clicking outside the modal.
        3. `Esc` key
- Background scroll disabled when modal is open.
- Clean and centered UI.

----------------------------------------

## Technologies Used

- HTML
- CSS
- Vanilla JavaScript (no libraries or frameworks)

---------------------------------------  

## Project Structure

- index.html
- style.css
- script.js
